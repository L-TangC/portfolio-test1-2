Interaction

Frist, I use obliqe rectangle on my website's header. This way will create more space for me to set the menu with other directions, right-angle way. It's benefitfor users to tab the menu more easy. While we browse a simple web as a portfolio wedsite, it'll be more convenient for users to click the menu items, without using the burger menu.
Second, I use diverse CSS effects on the categoray items. Like UI/UX page, the effect the I use is Parallax Scrollin. It's a great type to increase the quality of the preview boxs. In the other pages, Animation projects page and Editing projects page, the image will enlarge while ueser move their cursor on the preview blocks.
Last but not least, the "Contact me" page with popout effect, so the user can browse the info on anytime without changing pages.
